# 역할 분담 & 10시간 타임라인

## 5인 역할 분담

| 역할 | 담당 영역 | 산출물 | 의존성 |
|------|----------|--------|--------|
| **P1** Mock E-Accounting | REST 서버(:4000) + 시드데이터 + **관리자 웹화면**(카드내역 리스트, 접수 전표+결재라인 표시) | `/server`, `fixtures/` | 없음 — 가장 먼저 완성되어야 함 |
| **P2** 모바일 웹앱 | 촬영 UI, 영수증 보관함, 매칭 확인 화면, 전표 검토·상신 화면 | `/app` (모바일 뷰) | fixtures만 있으면 개발 가능 |
| **P3** OCR·크롭 파이프라인 | 이미지 업로드 → 자동 크롭 → Vision LLM 호출 → Receipt JSON. 데모 영수증 결과 캐시 | `/server/ocr` + `fixtures/receipts-ocr/` | P1 서버에 라우트 추가 |
| **P4** 매칭·전표 엔진 | 매칭 스코어링, 계정과목 자동분류, 전결라인 규칙엔진, 전표 초안 생성 | `/server/engine` | fixtures 기반 순수 함수로 개발 → 나중에 연결 |
| **P5** 출장모드·한도 + 통합/PM | 출장 등록·Cap 계산·환율·잔여한도 UI, 복지비 잔액 UI, **H6부터 통합 리드 + 데모 리허설** | `/app/trip`, 데모 스크립트 | P2와 같은 프론트 코드베이스 |

> 팀원 성향에 따라 조정하되, **P1은 속도 빠른 사람**, **P5는 전체 그림을 보는 사람(PM 겸)** 추천.
> P2·P5는 같은 프론트 리포에서 작업하므로 초반에 라우팅/컴포넌트 구조만 합의할 것.

## 기술 스택 (제안 — H0에 팀이 30분 안에 확정)

- 프론트: Vite + React (or Next.js) — 모바일 웹, Vercel 배포
- 백엔드: Node Express (or FastAPI) — 인메모리 저장 + JSON 파일. **DB 쓰지 말 것** (10시간에 사치)
- OCR: Claude API vision (프롬프트: "영수증에서 merchant/amount/currency/paidAt/items를 JSON으로")
- 모노레포: 이 리포 하나에 `/app`, `/server`, `/fixtures`, `/docs`

## 10시간 타임라인

| 시간 | 전원/개별 | 할 일 |
|------|-----------|-------|
| **H0–H1** | 전원 | 데모 시나리오 낭독·확정 → API 계약(02) 검토·동결 → 스택 확정 → 리포 스캐폴딩 → P1이 fixtures 커밋 |
| **H1–H6** | 개별 | 병렬 개발. 서로 기다리지 말고 fixtures로 개발. 막히면 그 기능을 데모에서 빼는 결정을 즉시(PM) |
| **H3** | 전원 15분 | 중간 체크인: 각자 화면/함수 시연. 빨간불이면 스코프 컷 |
| **H6–H8** | 전원 | **통합**: 업로드→OCR→매칭→전표→상신→관리자화면 E2E 1회 성공이 목표. P5 리드 |
| **H8–H9** | 전원 | 데모 시드데이터 정리, 데모용 영수증으로 실전 리허설, UI 다듬기 |
| **H9–H10** | 전원 | 리허설 2회 + 스크린 레코딩(백업) + 버퍼 |

## 통합이 안 터지게 하는 규칙 3개

1. **계약 우선**: API/JSON 스키마 바꾸려면 02 문서 먼저 고치고 전원에게 공지
2. **fixtures가 진실**: 모든 모듈은 fixtures 입출력으로 단독 실행 가능해야 함
3. **main은 항상 데모 가능**: 브랜치에서 작업, 동작 확인 후에만 머지. H6 이후 머지는 P5 승인

## 스코프 컷 우선순위 (시간이 모자랄 때 이 순서로 버린다)

1. 자동 크롭 (원본 이미지 그대로 → "크롭은 다음 버전" 멘트)
2. 복지비 보너스 장면 (데모 2:40 장면 삭제)
3. 매칭 "확인 필요" 중간 단계 (자동/미매칭 2단계로 단순화)
4. 환율 실시간 → 이미 고정 테이블이므로 영향 없음
5. **절대 안 버리는 것**: 촬영→OCR→매칭→전표 자동완성→상신→관리자 화면. 이게 데모의 척추.
