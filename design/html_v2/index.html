AXE-acc — 시제품#2 초기버전 포털

기존 Mock E-Accounting(E-acc)에 시제품#1 「찍으면 끝」을 머지한 업그레이드판 E-acc입니다. 기존 E-acc 기능(법인카드·출장비 정산, 전결규정 상신, 나의 문서함 등)은 전부 유지되고, 간편정산(영수증 촬영→OCR→자동매칭→전표) 기능과 모바일 창이 추가되었습니다. · 2026-07-22 · Main_2200_v2

🚀 실행 방법 (서버 1개로 PC + 모바일 모두)

  • cd server → npm install → npm start (포트 4000)
  • PC AXE-acc: http://localhost:4000/eaccounting/ — 상단 메뉴 「간편정산」
  • 모바일 AXE-acc: http://localhost:4000/eaccounting/m/ — 브라우저 모바일 뷰(F12) 권장
  • 루트 http://localhost:4000/는 팀 공용 접속 포털 — 이어카운팅(AXE-acc)·모바일웹(/app)·PC웹(/pc) 링크 3종
  • 시제품#1(별개 솔루션 버전)은 기존처럼 design/serve.js → :5173/5174 로 따로 확인 가능 — 서로 독립
  • 서버 없이 아래 링크(file://)로 열어도 내장 데모 데이터로 동작합니다 (실 API 연동은 서버 실행 시)
