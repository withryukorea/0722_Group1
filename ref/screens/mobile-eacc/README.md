# 기존 모바일 E-acc 화면 참고자료 (마스킹본)

실제 사내 모바일 e-Accounting 앱의 주요 화면 캡처 — AXE-acc 모바일 화면의 레이아웃·용어·IA 참고용.

**개인정보 마스킹 완료본만 커밋한다.** 사용자명은 홍길동, 결재자는 임의 가명(김한별·박두리·송세움·이가람·정누리),
사번·전화번호·전표번호·가맹점·금액·적요는 `**` 처리. 원본은 `ref/_originals/`(gitignore, 커밋 금지)에만 보관.

| 파일 | 화면 | 참고 포인트 |
|---|---|---|
| 01-홈 | 로그인 후 홈 | 인사말+소속 밴드, 업무 바로가기 2×2 카드(법인카드 n건·첨부파일·결재문서함 n건·대결자 지정) |
| 02-메뉴 | 햄버거 메뉴 | Home·법인카드·결재문서함·대결자 지정·첨부파일(등록/조회)·설정 + 시스템문의 |
| 03-법인카드-사용내역 | 사용내역 리스트 | 검색바, 건별 카드(가맹점/일시·사용자/금액), 하단 [현업완결 전표]·[회계팀 검토 전표] |
| 04-결재문서함-기안문서 | 문서함 3탭 | 결재할 문서/결재문서/기안문서 탭, 상태 필터, 결재완료·확정전표 뱃지, 결재선 날짜+이름 체인 |
| 05-전표-상세 | 전표 상세 | 정보표(문서명/기안일자/결재상태/기안의견), 결재선(기안·승인), [원문 보기] |
| 06-전표-원문 | 전표 원문(문서) | 결재정보 그리드, 전표정보, 카드이용내역 표(사용일자·금액·거래처·예산부서·계정과목·사용자·부가세·적요) |

AXE-acc 모바일에 반영할 만한 것: 문서함 3탭 구조(현재 우리는 2세그먼트), 대결자 지정 개념, 홈 바로가기 카드의 건수 배지, 현업완결/회계팀검토 전표 구분.
