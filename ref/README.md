# ref — 참고 자료 폴더

개발 과정에서 참고할 자료를 여기에 넣는다. 기획서 초안, 화면 레퍼런스 이미지, 경쟁 서비스 캡처, 스타일 가이드, 샘플 영수증, 외부 문서 등.

## 규칙

- **이 폴더의 파일은 참고용이다.** 코드에서 import 하거나 빌드에 포함시키지 않는다.
- 화면 작업을 시작하기 전에 이 폴더 내용을 먼저 확인한다.
- 자료가 늘어나면 하위 폴더로 분류한다. 예: `ref/screens/`, `ref/style/`, `ref/receipts/`
- 파일 이름은 무엇에 대한 참고인지 알 수 있게 적는다. 예: `ref/screens/토스-영수증-보관함.png`
